Object-oriented programming (OOP) is a way of programming that uses objects that encapsulate their own properties and methods. The main concepts of OOP are encapsulation, polymorphism, and inheritance. Polymorphism allows objects to override shared methods with a more specific implementation. All instances of a class or constructor function inherit all the properties and methods of its prototype.
